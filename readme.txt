== Description ==

This Wordpress plugin will manage your 16px ads on your Wordpress site. It even has a upload form to let your clients upload their own ads. The code for the plugin was originally done by Menguzar but i have taken up development since he has dropped the project.

Please visit http://vasthtml.com/js/wordpress-adicon-plugin/ for usage instructions.

For a DEMO visit www.vasthtml.com and scrol down to the footer. Feel free to upload any icon you have for free!

Be sure to backup your adicon “images” folder before you update.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `adicons` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Be sure to backup your adicon “images” folder before you update.

Please visit http://vasthtml.com/js/wordpress-adicon-plugin/ for usage instructions.
